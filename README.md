# 👻 Bruce's Ghostty 配置

**Claude Code 的最佳伴侣配置**  
零报错 · 左右分屏 · 一键放大 · 毛玻璃紫色主题

## 亮点
- **Cmd + D** 一键左右添加屏幕（写代码 + debug 两不误）
- **Cmd + Shift + Enter** 一键放大Claude输出
- Catppuccin Mocha 紫色毛玻璃
- Starship彩虹状态栏支持
- 布局永久保存

## 使用方法
1. 把 `config` 文件放到 `~/.config/ghostty/config`
2. 重载：**Cmd + Shift + ,**
3. 推荐搭配：`brew install --cask claude-code starship fastfetch btop`
